        ¿Qué es una función pura ?

        11
        pedronalbert
        24776 Puntos
        un año

        Es un concepto muy usado en la programacion funcional, una funcion pura basicamente es cuando cumple dos condiciones.

        • Dado los mismos parametros siempre retornará el mismo valor (esto queriendo decir que en cualquier momento que la llames con los mismos valores debe retornar siempre el mismo valor, la funcion no debe depender de otra cosa externa solo de los parametros que se le enviaron)
        • No produce side-effects (una explicación sencilla es que la funciona no debe alterar nada por fuera de su scope, ni realizar acciones que puedan variar con el tiempo como obtener la fecha actual, hacer una llamada http, alterar los parametros que le fueron enviados, obtener un número random)

        1
        devdanielgo

        Las devDependencies se utilizan sólamente durante el desarrollo de tu aplicación, como cuando levantas un servidor de archivos estáticos(como webpack-dev-server), o cuando compilas tu código css o javascript para generar el output(los archivos finales que se guardan dentro de dist).

        En dependencias vas a guardar los modulos que requiere tu aplicación para funcionar, por ejemplo si haces una aplicación que utilice bootstrap, guardarías esta dependencia en dependencies, en este caso estamos creando una aplicacion react, así que por esa razón react y react-dom se instalan como dependencies.

        var path = require("path");
var HtmlWebpackPlugin = require("html-webpack-plugin");

module.exports = {
  mode: "development",
  entry: "./src/index.js",
  output: {
    path: path.resolve(__dirname, "dist"),
    filename: "index_bundle.js"
  },
  module: {
    rules: [
      {
        test: /\.(js)$/,
        exclude: /node_modules/,
        use: {
          loader: "babel-loader",
          options: {
            presets: ["@babel/env", "@babel/react"]
          }
        }
      },
      { test: /\.css$/, use: ["style-loader", "css-loader"] },
      { test: /\.json$/, use: "json-loader" },
      {
        test: /\.(jpg|png|gif|jpeg|svg|woff|oet|ttf|mp4|webm)$/,
        use: {
          loader: "url-loader",
          options: {
            limit: 100000,
            name: "assets/[name].[hash].[ext]",
            output: {
              path: path.resolve(__dirname, "dist")
            }
          }
        }
      }
    ]
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: "public/index.html"
    })
  ]
};
